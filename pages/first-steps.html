Variables

Variable types

  • Numbers:
    let myAge = 22;
  • Strings:
    let myName = "Rodrigo Fernandes";
  • Booleans:
    let iAmAlive = true;
  • Arrays:
    let myNameCharacters = [7, 7, 2, 5, 9];
    Note: you can store different data types, even other arrays.
  • Objects:
    let myInfo = {name: myName, age: myAge, sex: "male"};
    To retrieve the name, for example, use
    myInfo.name;.

The typeof operator retrieves the data type of the variable/constant.

Numbers and operators

  • toFixed() : Rounds a number to a fixed amount of decimal places.
    let lotsOfDecimal = 1.766584958675746364;
    let twoDecimalPlaces = lotsOfDecimal.toFixed(2);
  • Number() : Converts a string into a number.
    myNumber = "256";
    Number(myNumber) + 3;
    The result will be 259.
  • Operators:
    + - * / % and **
  • Comparison operators:
    === !== < > <= and >=

Useful string methods

Strings as objects

Most things are objects in JavaScript. A string is one of those things.

  • length : Retrieves the length of the string.
    let browserType = "chrome";
    browserType.length;
  • browserType[x] : Retrieves a character in the x position.
  • indexOf() : Finds a substring index inside of the string. If the substring is not found, it returns -1.
    browserType.indexOf("ome"); // returns 3
  • slice() : Returns a substring based on the positions you feed it.
    browserType.slice(0,3); // returns "chr"
    browserType.slice(1); // returns "hrome"
  • toLowerCase() toUpperCase() : changes case.
    let os = "winDOWS";
    os.toLowerCase(); // returns "windows"
    os.toUpperCase(); // returns "WINDOWS"
  • replace() : replaces a substring.
    browserType.replace("chr", "d"); // returns "dome"
    Note: this does not change the actual value of browserType.

Arrays

Useful array methods

  • split() : separates a string into substrings and converts them into an array.
    let myCountries = "Brasil, Portugal, Angola";
    myCountries.split(", "); // returns ["Brasil", "Portugal", "Angola"]
  • join() : does the opposite.
    let myCities = ["Porto", "Lisboa", "Braga"];
    myCities.join(", "); // returns "Porto, Lisboa, Braga"

Adding and removing items

All of the methods below alter the actual array itself.

  • push() : adds an item to the end of the array.
    let myArray = ["USA", "England", "Spain"];
    myArray.push("France", "Netherlands"); // returns 5, the new length of the array
  • pop() : removes the last item of the array.
    myArray.pop(); // returns "Netherlands", the removed item
  • unshift() : adds an item to the beginning of the array.
    myArray.unshift("Canada"); // returns 5, the new length of the array
  • shift() : removes the first item of the array.
    myArray.shift(); // returns "Canada", the removed item
