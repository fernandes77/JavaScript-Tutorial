Variables

Variable types

  • Numbers:
    let myAge = 22;
  • Strings:
    let myName = "Rodrigo Fernandes";
  • Booleans:
    let iAmAlive = true;
  • Arrays:
    let myNameCharacters = [7, 7, 2, 5, 9];
  • Objects:
    let myInfo = {name: myName, age: myAge, sex: "male"};
    To retrieve the name, for example, use
    myInfo.name;.

The typeof operator retrieves the data type of the variable/constant.
