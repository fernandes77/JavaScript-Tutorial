Conditionals and Loops

Conditionals and loops in JavaScript work just like in C.

If...else

  • if (condition 1) {
    // code to run if condition 1 is true
    }
    else if (condition 2) {
    // code to run instead if condition 2 is true
    }
    else {
    // code to run instead
    }
  • Ternary operators:
    ( condition ) ? // code to run : // code to run instead ;

Switch statements

These are used when there are too many conditions.

switch (expression) {
case choice1:
// code to run
break;

case choice2:
// code to run instead
break;

default:
// actually, just run this code
}

Looping code

There are 3 ways of looping code. for, while and do...while.

  • for (initializer; condition; final-expression) {
    // code to run
    }
  • while (condition) {
    // code to run

    final-expression
    }
  • do {
    // code to run

    final-expression
    } while (condition);

Functions

Functions versus methods

Functions that are part of objects are called methods.

Custom functions

function name(parameters) {
// code to run
// return something (optional)
}

Anonymous functions

Anonymous functions are functions that have no name. It is generally used with an event handler.

const myButton = document.querySelector("button");

myButton.onClick = function() {
alert("hello");
}

You can also assign it to a variable.

const myGreeting = function() {
alert("hello");
}

Function scope

In JavaScript functions have their own local scope, which can't be used anywhere else. Outside of the functions there is the global scope.

Making a function

Note: if you put the parentheses when you assign the function to an event you will call that function immediately. Do not include the parentheses:

btn.onclick = displayMessage;


Introduction to events

Events are occurrences that happen in the system you are programming, which the system tells you about so you can respond to them in some way if desired.

Event handlers

Some examples are:

  • btn.onclick: when the button is clicked;
  • btn.onfocus: when the button is focused on with tab;
  • btn.ondblclick: when the button is double-clicked;
  • window.onkeyup: when the UP key is pressed;
  • btn.onmouseover: when the mouse pointer hovers over the button.

addEventListener

The addEventListener() function works like event handlers but has some advantages over them.

  • You can remove an added listener with removeEventListener();
  • You can add multiple functions to an event.

Event objects

You might encounter a parameter specified with a name such as e, evt or event. You can then use event.target.style.backgroundColor instead of a variable created by querySelector.

function bgChange(event) {
var rndCol =
"rgb(" + random(255) + "," + random(255) + "," + random(255) + ")";
event.target.style.backgroundColor = rndCol;
}

square.addEventListener("click", bgChange);

Event bubbling

When an element and it's parent element have event listeners and the child's event occurs, both event listeners will be activated.
To prevent this, use stopPropagation() so that only the child's event listener is activated.

video.onclick = function(e) {
e.stopPropagation();
video.play();
};
